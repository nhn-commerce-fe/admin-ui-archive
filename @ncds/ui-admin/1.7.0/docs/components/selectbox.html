SelectBox

커스텀 셀렉트 박스 컴포넌트입니다. 화면 위치에 따라 옵션이 위/아래로 동적으로 표시됩니다. 외부 클릭 감지, 가상 포커스 기반 키보드 네비게이션, react-hook-form 지원 등 다양한 기능을 제공합니다.

기본 사용법​

import { SelectBox, type OptionType } from '@ncds/ui-admin';

const options: OptionType[] = [
  { id: 1, label: '옵션 1' },
  { id: 2, label: '옵션 2' },
  { id: 3, label: '옵션 3' },
];

function App() {
  const [value, setValue] = useState<string | number>('');

  return <SelectBox placeholder="옵션을 선택하세요" optionItems={options} value={value} onChange={setValue} />;
}

키보드 조작​

  • Enter / Space: 드롭다운 열기/닫기, 옵션 선택
  • Arrow Down / Arrow Up: 옵션 간 이동
  • Home: 첫 번째 옵션으로 이동
  • End: 마지막 옵션으로 이동
  • 문자 키: 해당 문자로 시작하는 옵션 검색
  • Escape: 드롭다운 닫기

접근성​

  • role="combobox" 및 role="listbox" 사용
  • aria-expanded, aria-haspopup 속성으로 상태 표시
  • aria-activedescendant 속성으로 현재 포커스된 옵션 표시
  • aria-selected 속성으로 선택된 옵션 상태 표시
  • 가상 포커스 시스템으로 부드러운 키보드 네비게이션 지원
  • 키보드와 마우스 간 자연스러운 전환 지원

위치 감지 로직​

SelectBox는 화면에서의 위치를 자동으로 감지하여 드롭다운 방향을 결정합니다:

  • 일반적인 경우: 드롭다운이 아래쪽으로 표시
  • 화면 아래쪽에 위치한 경우: 드롭다운이 위쪽으로 표시

HTML 버전​

React를 사용하지 않는 환경에서도 SelectBox를 사용할 수 있습니다.

기본 사용법​

1. 기존 select와 연동​

<select> 태그와 연동하여 사용할 수 있습니다. 여러 가지 방법이 있습니다:

방법 1: data-target 속성 사용

<select name="city" id="city-select" style="display: none">
  <option value="">도시를 선택하세요</option>
  <option value="seoul" selected>서울특별시</option>
  <option value="busan">부산광역시</option>
  <option value="daegu">대구광역시</option>
</select>

<!-- 커스텀 SelectBox -->
<div id="my-selectbox" data-target="city-select"></div>

<script>
  document.addEventListener('DOMContentLoaded', function () {
    // SelectBox 초기화
    const selectBox = new ncua.SelectBox(document.getElementById('my-selectbox'), {
      size: 'md', // 'xs' | 'sm' | 'md'
      placeholder: '도시를 선택하세요',
    });
  });
</script>

방법 2: select를 자식으로 포함

<div id="my-selectbox">
  <select name="city" id="city-select" style="display: none">
    <option value="">도시를 선택하세요</option>
    <option value="seoul" selected>서울특별시</option>
    <option value="busan">부산광역시</option>
    <option value="daegu">대구광역시</option>
  </select>
</div>

<script>
  document.addEventListener('DOMContentLoaded', function () {
    const selectBox = new ncua.SelectBox(document.getElementById('my-selectbox'), {
      size: 'md',
    });
  });
</script>

방법 3: select 태그 자체를 SelectBox로 변환

2. JavaScript 옵션 직접 설정​

JavaScript에서 직접 옵션을 설정할 수도 있습니다. 옵션은 {id, label} 또는 {id, text} 형식을 사용할 수 있습니다.

<div id="js-selectbox"></div>

<script>
  document.addEventListener('DOMContentLoaded', function () {
    const selectBox = new ncua.SelectBox(document.getElementById('js-selectbox'), {
      options: [
        { id: 1, label: '옵션 1' }, // 또는 { id: 1, text: '옵션 1' }
        { id: 2, label: '옵션 2' },
        { id: 3, label: '옵션 3' },
      ],
      value: 1, // 초기 선택값
      placeholder: '옵션을 선택하세요',
      size: 'md',
      onChange: (value) => {
        console.log('선택된 값:', value);
      },
    });
  });
</script>

3. Multiple 선택 모드​

Multiple 선택 모드를 사용할 수 있습니다.

<div id="multiple-selectbox"></div>

<script>
  document.addEventListener('DOMContentLoaded', function () {
    const multipleSelectBox = new ncua.SelectBox(document.getElementById('multiple-selectbox'), {
      options: [
        { id: 'react', label: 'React' },
        { id: 'vue', label: 'Vue.js' },
        { id: 'angular', label: 'Angular' },
        { id: 'svelte', label: 'Svelte' },
      ],
      value: ['react'], // 초기 선택값 (배열)
      placeholder: '프레임워크를 선택하세요',
      multiple: true, // 다중 선택 모드
      showFooterButtons: true, // 하단 버튼 표시
      size: 'md',
      onChange: (values) => {
        console.log('선택된 값들:', values);
      },
      onEdit: () => {
        console.log('편집 버튼 클릭');
      },
      onSelectAll: () => {
        console.log('전체 선택 버튼 클릭');
      },
      onComplete: () => {
        console.log('완료 버튼 클릭');
      },
    });
  });
</script>

메서드​

SelectBox 인스턴스에서 사용할 수 있는 메서드들입니다.

const selectBox = new ncua.SelectBox(element, options);

// 값 설정 (옵션 객체 배열 사용)
selectBox.setValues([{ id: 'seoul', label: '서울특별시' }]);

// Multiple 선택 모드에서 값 설정
selectBox.setValues([
  { id: 'seoul', label: '서울특별시' },
  { id: 'busan', label: '부산광역시' },
]);

// 현재 선택된 옵션들 가져오기 (옵션 객체 배열 반환)
const selectedOptions = selectBox.getValues(); // Array<{id, label, ...}>

// 옵션 목록 가져오기
const allOptions = selectBox.getOptions(); // Array<{id, label, ...}>

// 옵션 목록 업데이트
selectBox.updateOptions([
  { id: 1, label: '새 옵션 1' },
  { id: 2, label: '새 옵션 2' },
]);

// 활성화/비활성화
selectBox.setDisabled(true);
selectBox.setDisabled(false);

// 열기/닫기
selectBox.open();
selectBox.close();

// 포커스 설정
selectBox.focus();

// Multiple 모드 전용 메서드들
selectBox.selectAll(); // 전체 선택
selectBox.deselectAll(); // 전체 선택 해제
selectBox.toggleSelectAll(); // 전체 선택 토글
const count = selectBox.getSelectedCount(); // 선택된 항목 개수
const isMultiple = selectBox.isMultiple(); // Multiple 모드 여부

// 컴포넌트 제거
selectBox.destroy();
