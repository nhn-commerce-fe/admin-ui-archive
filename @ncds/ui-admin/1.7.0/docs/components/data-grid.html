DataGrid

DataGrid는 기존 Table 등 원자/분자 컴포넌트를 레이아웃 규칙에 맞게 조합하는 컨테이너 컴포넌트입니다.
조회 결과형, 일반 가로형, 일반 세로형 세 가지 유형을 제공합니다.

DataGrid는 반드시 Block 또는 Tab 구조 내에 포함되어야 합니다.

function App() {
  return (
    <DataGrid variant="search-result">
      <DataGrid.Summary>검색 5건 / 전체 128건</DataGrid.Summary>
      <DataGrid.Table selectable hoverable>
        <Table.Header>
          <Table.Row>
            <Table.HeaderCell>ID</Table.HeaderCell>
            <Table.HeaderCell>상품명</Table.HeaderCell>

DataGrid.Table​

PropTypeDefaultDescription
type"horizontal" | "vertical"variant에 따라 자동테이블 유형 (명시 시 우선 적용)
fixedHeaderbooleanfalse헤더 고정 여부
maxHeightstring | number-Fixed Header 시 최대 높이
hoverableboolean-행 호버 효과
selectableboolean-선택 모드

DataGrid.ActionBar​

PropTypeDefaultDescription
position"top" | "bottom"'top'테이블 상단/하단 위치
align"left" | "right" | "space-between"'space-between'내부 콘텐츠 정렬
