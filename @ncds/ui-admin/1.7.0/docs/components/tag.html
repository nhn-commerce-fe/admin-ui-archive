Tag

태그 컴포넌트는 정보를 분류하고 필터링하기 위한 시각적 표시를 제공합니다. 레이블, 카테고리, 필터 등을 나타내는 데 사용되며, 아이콘, 체크박스, 닫기 버튼 등의 기능을 포함할 수 있습니다.

인터랙티브 예제​

Tag10
sizeselect
태그의 크기를 선택합니다.
iconpreset
태그의 아이콘을 선택합니다.
textstring
태그의 텍스트를 입력합니다.
countstring
태그의 카운트를 입력합니다.
closeboolean
태그의 닫기 버튼을 표시합니다.
maxLengthnumber
텍스트 최대 길이를 설정합니다. (기본값: 20)

기본 사용법​

import { Tag } from '@ncds/ui-admin';

function App() {
  return <Tag text="태그" />;
}

HTML 사용법​

HTML에서 직접 사용할 수 있습니다. ncua.Tag 클래스를 사용하여 태그를 생성하고 DOM에 추가합니다.

기본 사용법​

<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <title>Tag Example</title>
    <link rel="stylesheet" href="path/to/ui-admin.css" />
  </head>
  <body>
    <div id="tag-container"></div>

    <script src="path/to/ui-admin.bundle.js"></script>
    <script>
      // 기본 태그 생성
      const tag = new ncua.Tag({ text: '기본 태그' });
      document.getElementById('tag-container').appendChild(tag.getElement());
    </script>
  </body>
</html>

아이콘과 카운트가 있는 태그​

<script>
  // 아이콘과 카운트가 있는 태그
  const tagWithIcon = new ncua.Tag({
    text: '상품',
    count: '15',
    icon: { type: 'icon', icon: 'arrow-left', color: 'black' },
    size: 'md',
  });
  document.getElementById('tag-container').appendChild(tagWithIcon.getElement());

  // 닫기 버튼이 있는 태그
  const closeableTag = new ncua.Tag({
    text: '삭제 가능 태그',
    close: true,
    onButtonClick: (e) => {
      e.target.closest('.ncua-tag').remove();
    },
  });
  document.getElementById('tag-container').appendChild(closeableTag.getElement());
</script>
