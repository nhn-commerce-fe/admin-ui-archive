Table

Table은 어드민 환경에서 데이터를 구조화하여 표시하는 핵심 컴포넌트입니다.
Horizontal Table(목록형)과 Vertical Table(상세형) 두 가지 유형을 제공합니다.

인터랙티브 예제​

아래 UI에서 다양한 속성을 직접 변경하여 Table 컴포넌트의 모습을 확인해보세요.

ID상품명카테고리가격상태
1상품 A전자기기₩50,000판매중
2상품 B의류₩30,000품절
3상품 C식품₩15,000판매중
4상품 D전자기기₩120,000배송중
5상품 E생활용품₩8,500판매중

기본 사용법​

import { Table } from '@ncds/ui-admin';

function App() {
  return (
    <Table type="horizontal">
      <Table.Header>
        <Table.Row>
          <Table.HeaderCell>ID</Table.HeaderCell>
          <Table.HeaderCell>이름</Table.HeaderCell>
          <Table.HeaderCell>상태</Table.HeaderCell>
        </Table.Row>
      </Table.Header>
      <Table.Body>
        <Table.Row>
          <Table.Cell>1</Table.Cell>
          <Table.Cell>상품 A</Table.Cell>
          <Table.Cell>판매중</Table.Cell>
        </Table.Row>
      </Table.Body>
    </Table>
  );
}

Vertical Table (상세형)​

단일 객체의 속성을 Label-Value 쌍으로 표시하는 상세형 테이블입니다.
필수 항목은 Label 앞에 * 표시를 추가합니다.

*상품명프리미엄 노트북
*카테고리전자기기
*가격₩1,500,000
상태판매중
설명고성능 프로세서와 16GB RAM을 탑재한 프리미엄 노트북입니다.

Empty State (빈 상태)​

데이터가 없을 때 Table.Empty를 사용하여 빈 상태 메시지를 표시합니다.

ID상품명카테고리가격상태
등록된 게시물이 없습니다.

Props​

Table​

PropTypeDefaultDescription
type"horizontal" | "vertical"'horizontal'테이블 유형
fixedHeaderbooleanfalse헤더 고정 여부
maxHeightstring | number-Fixed Header 시 스크롤 영역 최대 높이
hoverablebooleantrue행 호버 효과 활성화
selectablebooleanfalse체크박스 선택 모드 (하단 여백 추가)
classNamestring-추가 CSS 클래스

Table.Row​

PropTypeDefaultDescription
selectedbooleanfalse행 선택 상태 (좌측 indicator bar 표시)
status"warning" | "error"-행 상태 (배경색 변경)

Table.HeaderCell​

PropTypeDefaultDescription
sortDirection"asc" | "desc" | "none"-정렬 방향 (onSort와 함께 사용)
onSort() => void-정렬 클릭 핸들러
widthstring | number-컬럼 너비

Table.ColGroup​

PropTypeDefaultDescription
widths(string | number)[]-각 컬럼의 너비 배열

Table.Footer​

PropTypeDefaultDescription
childrenReactNode-Footer 영역에 표시할 콘텐츠
classNamestring-추가 CSS 클래스

Table.Empty​

PropTypeDefaultDescription
colSpannumber-빈 상태 셀이 차지할 컬럼 수
childrenReactNode'등록된 게시물이 없습니다.'빈 상태 메시지
