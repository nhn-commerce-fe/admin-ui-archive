Notification

Notification은 사용자에게 중요한 정보나 상태 변경을 알려주는 컴포넌트입니다. 알림은 세 가지 유형으로 제공됩니다: 측면에 떠있는 형태의 'Floating' 타입, 전체 너비를 차지하는 'Full-width' 타입, 그리고 메시지 형태의 'Message' 타입.

인터랙티브 예제​

아래 UI에서 다양한 속성을 직접 변경하여 알림 컴포넌트의 모습을 확인해보세요.

}

유형​

Notification 컴포넌트는 floating, full-width, message 세 가지 유형을 지원합니다. floating 유형은 측면에 떠 있는 형태의 알림이며, full-width 유형은 페이지 상단을 가로지르는 전체 너비의 알림입니다. message 유형은 메시지 형태의 알림으로, 액션 버튼과 다시보지 않기 기능을 제공합니다.

색상 변형​

Notification 컴포넌트는 다섯 가지 색상 변형을 지원합니다: neutral, error, warning, success, info. 각 변형은 다른 상황이나 메시지의 중요도를 시각적으로 구분하는 데 도움이 됩니다. info 색상은 full-width 타입에서만 지원되며, message 타입은 neutral, error, warning, success 네 가지 색상만 지원합니다.

액션 버튼​

알림에 추가 액션을 위한 버튼을 추가할 수 있습니다. actions prop을 사용하여 버튼 등의 액션 요소를 전달할 수 있습니다.

액션 버튼 사용 예제​

import { Notification } from '@ncds/ui-admin';

function App() {
  return (
    <Notification
      title="액션 버튼이 있는 알림"
      supportingText="사용자가 추가 조치를 취할 수 있는 버튼이 있는 알림입니다."
      onClose={() => console.log('닫기 버튼 클릭')}
      actions={[
        { label: '확인', onClick: () => console.log('확인') },
        { label: '취소', onClick: () => console.log('취소') },
      ]}
    />
  );
}

링크 및 다시보지 않기​

Full-width 타입의 알림에서는 supportTextLink prop을 사용하여 supportingText에 링크를 추가할 수 있습니다. Full-width와 Message 타입의 알림에서는 onHidePermanently prop을 사용하여 다시보지 않기 기능을 제공할 수 있습니다.

NotificationAction​

알림에 액션 버튼을 추가할 때 다음과 같은 구조의 객체를 사용할 수 있습니다:

interface NotificationAction {
  /**
   * 액션 버튼 텍스트
   */
  label: string;
  /**
   * 액션 버튼 클릭 이벤트 핸들러
   */
  onClick?: () => void;
  /**
   * 액션 버튼 색상 (2가지만 지원)
   */
  hierarchy?: 'text-gray' | 'text';
}

액션 버튼 배열 사용 예제​

import { Notification } from '@ncds/ui-admin';

function App() {
  // NotificationAction 배열 정의
  const actions = [
    {
      label: '확인',
      onClick: () => console.log('확인 버튼 클릭'),
    },
    {
      label: '취소',
      onClick: () => console.log('취소 버튼 클릭'),
    },
  ];

  return (
    <Notification
      title="액션 버튼이 있는 알림"
      supportingText="액션 버튼 배열을 사용한 알림입니다."
      color="success"
      actions={actions}
    />
  );
}

                포인트 충전
              </button>
            </div>
            <button
              class="ncua-notification__action-button ncua-notification__action-button--text ncua-message-notification__hide-link"
            >
              다시 보지 않기
            </button>
            <button class="ncua-message-notification__close-button" type="button">
              <!-- 닫기 버튼 아이콘 -->
            </button>
          </div>
        </div>
      </div>
    </div>
  </body>
</html>

2. JavaScript 클래스를 사용하는 방법​

JavaScript API를 사용하여 동적으로 알림을 생성하고 제어하는 방법입니다.

아래는 HTML 파일에서 바로 사용할 수 있는 완전한 예제입니다:

<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>NCDS Notification 예제</title>
    <!-- NCDS UI Admin CSS -->
    <link rel="stylesheet" href="https://fe-sdk.cdn-nhncommerce.com/@ncds/ui-admin/0.0/main.min.css" />
  </head>
  <body>
    <div style="padding: 20px;">
      <h1>Notification 예제</h1>

      <div style="margin-bottom: 20px;">
        <h3>기본 알림</h3>
        <button id="basic-btn">기본 알림 표시</button>
      </div>

      <div style="margin-bottom: 20px;">
        <h3>색상별 알림</h3>
        <button id="success-btn">성공</button>
        <button id="error-btn">오류</button>
        <button id="warning-btn">경고</button>
        <button id="info-btn">정보</button>
      </div>

      <div style="margin-bottom: 20px;">
        <h3>Full-width 알림</h3>
        <button id="fullwidth-btn">전체 너비 알림</button>
        <button id="fullwidth-link-btn">링크가 있는 전체 너비 알림</button>
        <button id="fullwidth-hide-btn">다시보지 않기가 있는 전체 너비 알림</button>
      </div>

      <div style="margin-bottom: 20px;">
        <h3>Message 알림</h3>
        <button id="message-btn">기본 Message 알림</button>
        <button id="message-actions-btn">액션 버튼이 있는 Message 알림</button>
        <button id="message-hide-btn">다시보지 않기가 있는 Message 알림</button>
      </div>

      <div style="margin-bottom: 20px;">
        <h3>액션 버튼 알림</h3>
        <button id="action-btn">액션 버튼 알림</button>
      </div>

      <div style="margin-bottom: 20px;">
        <h3>자동 닫기 알림</h3>
        <button id="autoclose-btn">3초 후 자동 닫기</button>
        <button id="permanent-btn">영구 알림</button>
      </div>
    </div>

    <!-- NCDS UI Admin JavaScript -->
    <script src="https://fe-sdk.cdn-nhncommerce.com/@ncds/ui-admin/0.0/main.min.js"></script>

    <script>
      document.addEventListener('DOMContentLoaded', function () {
        // 기본 알림
        document.getElementById('basic-btn').addEventListener('click', function () {
          const notification = new ncua.Notification({
            title: '기본 알림',
            supportingText: '이것은 기본 알림입니다.',
            onClose: () => console.log('알림 닫기'),
          });
          notification.show();
        });

        // 성공 알림
        document.getElementById('success-btn').addEventListener('click', function () {
          ncua.Notification.success('성공!', '작업이 완료되었습니다.');
        });

        // 오류 알림
        document.getElementById('error-btn').addEventListener('click', function () {
          ncua.Notification.error('오류 발생', '문제가 발생했습니다.');
        });

        // 경고 알림
        document.getElementById('warning-btn').addEventListener('click', function () {
          ncua.Notification.warning('주의', '주의가 필요한 상황입니다.');
        });

        // 정보 알림
        document.getElementById('info-btn').addEventListener('click', function () {
          ncua.Notification.info('정보', '새로운 정보가 있습니다.');
        });

        // Full-width 알림
        document.getElementById('fullwidth-btn').addEventListener('click', function () {
          const notification = ncua.Notification.showFullWidth({
            title: '전체 너비 알림',
            supportingText: '중요한 공지사항입니다.',
            color: 'warning',
            onClose: () => console.log('전체 너비 알림 닫기'),
          });
          notification.show();
        });

        // Full-width 알림 (링크 포함)
        document.getElementById('fullwidth-link-btn').addEventListener('click', function () {
          const notification = ncua.Notification.showFullWidth({
            title: '중요 공지',
            supportingText: '자세한 내용은 링크를 확인하세요.',
            color: 'info',
            supportTextLink: 'https://example.com',
            onClose: () => console.log('링크가 있는 전체 너비 알림 닫기'),
          });
          notification.show();
        });

        // Full-width 알림 (다시보지 않기)
        document.getElementById('fullwidth-hide-btn').addEventListener('click', function () {
          const notification = ncua.Notification.showFullWidth({
            title: '시스템 공지',
            supportingText: '이 알림을 다시 보지 않으려면 "다시보지 않기" 버튼을 클릭하세요.',
            color: 'warning',
            onHidePermanently: () => {
              console.log('다시보지 않기 클릭');
              // 로컬 스토리지에 저장하거나 서버에 요청하는 로직
              localStorage.setItem('notification-hidden', 'true');
            },
            onClose: () => console.log('다시보지 않기가 있는 전체 너비 알림 닫기'),
          });
          notification.show();
        });

        // Message 알림
        document.getElementById('message-btn').addEventListener('click', function () {
          const notification = new ncua.Notification({
            type: 'message',
            title: '고도몰이 새로워졌습니다.',
            supportingText: '포인트가 소진되는 경우, 메시지 및 자동알림 발송이 안되므로 포인트를 충전하시기 바랍니다.',
            color: 'neutral',
            onClose: () => console.log('Message 알림 닫기'),
          });
          notification.show();
        });

        // Message 알림 (액션 버튼 포함)
        document.getElementById('message-actions-btn').addEventListener('click', function () {
          const notification = new ncua.Notification({
            type: 'message',
            title: '고도몰이 새로워졌습니다.',
            supportingText: '포인트가 소진되는 경우, 메시지 및 자동알림 발송이 안되므로 포인트를 충전하시기 바랍니다.',
            color: 'neutral',
            actions: [
              {
                label: '조회하기',
                hierarchy: 'text',
                onClick: () => {
                  console.log('조회하기 클릭');
                  notification.remove();
                },
              },
              {
                label: '포인트 충전',
                hierarchy: 'text-gray',
                onClick: () => {
                  console.log('포인트 충전 클릭');
                  notification.remove();
                },
              },
            ],
            onClose: () => console.log('Message 액션 알림 닫기'),
          });
          notification.show();
        });

        // Message 알림 (다시보지 않기)
        document.getElementById('message-hide-btn').addEventListener('click', function () {
          const notification = new ncua.Notification({
            type: 'message',
            title: '시스템 공지',
            supportingText: '이 알림을 다시 보지 않으려면 "다시 보지 않기" 버튼을 클릭하세요.',
            color: 'warning',
            onHidePermanently: () => {
              console.log('다시보지 않기 클릭 (Message)');
              // 로컬 스토리지에 저장하거나 서버에 요청하는 로직
              localStorage.setItem('message-notification-hidden', 'true');
            },
            onClose: () => console.log('다시보지 않기가 있는 Message 알림 닫기'),
          });
          notification.show();
        });

        // 액션 버튼 알림
        document.getElementById('action-btn').addEventListener('click', function () {
          const notification = new ncua.Notification({
            title: '액션이 있는 알림',
            supportingText: '확인 또는 취소를 선택하세요.',
            color: 'neutral',
            actions: [
              {
                label: '취소',
                hierarchy: 'text-gray',
                onClick: () => {
                  console.log('취소 클릭');
                  notification.remove();
                },
              },
              {
                label: '확인',
                hierarchy: 'text',
                onClick: () => {
                  console.log('확인 클릭');
                  notification.remove();
                },
              },
            ],
            onClose: () => console.log('액션 알림 닫기'),
          });
          notification.show();
        });

        // 자동 닫기 알림
        document.getElementById('autoclose-btn').addEventListener('click', function () {
          const notification = new ncua.Notification({
            type: 'floating',
            title: '자동 닫기 알림',
            supportingText: '3초 후 자동으로 사라집니다.',
            color: 'success',
            autoClose: 3000, // 3초 후 자동 닫기
            onClose: () => console.log('자동으로 닫혔습니다.'),
          });
          notification.show();
        });

        // 영구 알림
        document.getElementById('permanent-btn').addEventListener('click', function () {
          const notification = new ncua.Notification({
            type: 'floating',
            title: '영구 알림',
            supportingText: '수동으로만 닫을 수 있습니다. X 버튼을 클릭하세요.',
            color: 'neutral',
            autoClose: 0, // 자동으로 닫히지 않음
            onClose: () => console.log('수동으로 닫혔습니다.'),
          });
          notification.show();
        });
      });
    </script>
  </body>
</html>
