아이콘

패키지 설치​

npm install @ncds/ui-admin-icon

사용 방법​

아이콘 컴포넌트​

아이콘 컴포넌트를 아래와 같이 호출할 수 있습니다:

import { Plus, Heart } from '@ncds/ui-admin-icon';

function MyComponent() {
  return (
    <div>
      <Plus />
      <Heart width={24} height={24} color="red" />
    </div>
  );
}

동적 아이콘​

동적 아이콘 컴포넌트는 필요에 따라 아이콘을 로드하는 컴포넌트입니다. 아이콘 이름을 전달하면 해당 아이콘을 로드합니다:

빌드시 모든 아이콘이 청크로 분리되어 번들링되며, 런타임에 필요한 아이콘만 로드됩니다.

import Icon from '@ncds/ui-admin-icon/dynamic';

function MyComponent() {
  return (
    <div>
      <Icon name="plus" />
      <Icon name="heart" width={24} height={24} color="red" />
    </div>
  );
}

name 속성은 패키지에서 타입이 정의되어 있어 타입 체크에 활용할 수 있습니다:

import type { IconName } from '@ncds/ui-admin-icon/dynamic';

const iconName: IconName = 'plus';

Props​

PropTypeDefaultDescription
nameIconName-사용할 아이콘의 이름
widthnumber | string24아이콘의 너비
heightnumber | string24아이콘의 높이
colorstring-아이콘의 색상

위 Props 외에도 SVGSVGElement의 모든 속성을 지원합니다.
