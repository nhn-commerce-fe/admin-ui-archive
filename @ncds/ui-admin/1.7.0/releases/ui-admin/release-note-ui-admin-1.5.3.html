v1.5.3

2026년 2월 24일

ui-admin 릴리즈 노트 (v1.5.3)​

신규 기능​

  • imageFileInput 컴포넌트의 스타일을 추가하였습니다.
    • 오류 상태일때 보여줄 수 있는 UI 스타일을 추가하였습니다.
  • RangeDatePickerWithButtons 컴포넌트에 periodItems prop을 추가하였습니다.
    • 이 prop을 사용하면 PERIOD_ITEM 대신 프로젝트에 맞는 커스텀 기간 항목을 정의할 수 있습니다.

기능 개선​

  • HorizontalTab 컴포넌트의 사이즈가 변경 안되는 이슈 수정했습니다.
  • PasswordInput 컴포넌트의 onChange 이벤트가 동작 안되는 이슈 수정했습니다.
  • DatePicker 컴포넌트의 선택날짜가 이전, 다음달일 경우 디자인 가이드와 안맞는 색상으로 표시되는 부분 수정했습니다.

스타일 개선​

  • FileInput 컴포넌트의 힌트 스타일을 기존 1.5.1 버전 이하의 스타일로 복구하였습니다.
