v1.6.0

2026년 3월 3일

ui-admin 릴리즈 노트 (v1.6.0)​

Breaking Changes​

아이콘 시스템 변경​

아이콘을 문자열 이름('search-lg')으로 전달하던 방식에서, 컴포넌트를 직접 import하여 전달하는 방식으로 변경되었습니다.

변경 전 (v1.5.x)

import { Button } from '@ncds/ui-admin';

<Button label="검색" leadingIcon={{ type: 'icon', icon: 'search-lg' }} />;

변경 후 (v1.6.0)

import { Button } from '@ncds/ui-admin';
import { SearchLg } from '@ncds/ui-admin-icon';

<Button label="검색" leadingIcon={{ type: 'icon', icon: SearchLg }} />;

마이그레이션 방법:

  1. @ncds/ui-admin-icon를 의존성으로 추가합니다.
  2. icon prop에 문자열 대신 import한 컴포넌트를 전달합니다. (예: 'search-lg' → SearchLg, 'arrow-left' → ArrowLeft)

이 변경은 SideSlotType의 icon 필드를 사용하는 모든 컴포넌트에 적용됩니다: Button, ButtonGroup, Dropdown, SelectBox, ComboBox, Modal, Notification, FeaturedIcon, Tag 등.

아이콘 타입 변경​

아이콘을 prop으로 받는 컴포넌트의 타입이 변경되었습니다.

항목변경 전변경 후
SideSlotType의 icon 필드IconName (문자열)SlotIconComponent (컴포넌트)
아이콘 단독 prop (Dropdown, Modal 등)stringSlotIconComponent

SlotIconComponent 타입이 필요하면 @ncds/ui-admin에서 import할 수 있습니다.

import type { SlotIconComponent, SideSlotType } from '@ncds/ui-admin';

신규 기능​

  • SelectBox 컴포넌트에 autoWidth, align prop을 추가하였습니다.
  • SelectBox 옵션에 icon 필드를 추가하였습니다.
