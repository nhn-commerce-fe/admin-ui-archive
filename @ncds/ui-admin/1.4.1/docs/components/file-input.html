File Input

FileInput 컴포넌트는 사용자로부터 파일을 업로드하기 위한 UI 요소입니다. 이 컴포넌트는 다양한 파일 타입을 지원하며, 파일 크기 제한, 개수 제한 등의 기능을 제공합니다.

인터랙티브 예제​

파일 입력 필드​

파일 입력을 위한 특수 컴포넌트의 속성을 변경하여 확인해보세요.

  • 파일은 1개씩 첨부 가능하며 최대 첨부 가능한 파일은 2개입니다.
  • 파일 용량 5MB 이내이며 첨부 가능한 형식은 gif, png, jpeg, jpg, pdf 입니다.

UploadedFile​

기본 사용법 (onChange + Tag 클래스)​

onChange를 사용하여 파일을 받아, Tag 클래스로 직접 태그를 생성하는 예제입니다. FileInput은 파일 선택만 담당하고, 태그 렌더링은 사용자가 직접 처리합니다.

// Controlled mode로 사용
let currentFiles = [];
const tagInstances = new Map(); // 파일명과 Tag 인스턴스 매핑

const fileInput = new ncua.FileInput({
  container: 'file-input-container',
  label: '파일 업로드 (사용자 직접 태그 생성)',
  buttonLabel: '파일 찾기',
  multiple: true,
  onChange: (newFiles) => {
    // 새로 선택된 파일들을 currentFiles에 추가
    currentFiles = [...currentFiles, ...newFiles];

    // Tag 클래스를 사용하여 태그 생성
    const container = document.getElementById('tag-container');
    newFiles.forEach((file) => {
      const tag = new ncua.Tag({
        text: file.name,
        size: 'sm',
        close: true,
        onButtonClick: () => {
          // 태그 제거
          tag.destroy();
          tag.getElement().remove();
          // 파일 목록에서 제거
          currentFiles = currentFiles.filter((f) => f.name !== file.name);
          // 태그 인스턴스 맵에서도 제거
          tagInstances.delete(file.name);
        },
      });

      // 태그 인스턴스 저장
      tagInstances.set(file.name, tag);

      // 컨테이너에 태그 추가
      container.appendChild(tag.getElement());
    });
  },
});
