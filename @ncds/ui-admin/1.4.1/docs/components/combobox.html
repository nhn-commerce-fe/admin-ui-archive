ComboBox

검색 기능이 있는 드롭다운 선택 컴포넌트입니다. 사용자가 입력 필드에 검색어를 입력하면 옵션 목록이 필터링되고, 클릭하여 드롭다운을 열어 원하는 옵션을 선택할 수 있습니다.

인터렉티브 예제​

입력 필드에 표시되는 플레이스홀더 텍스트

입력 필드 위에 표시되는 라벨 텍스트

필수 입력 여부 (라벨에 * 표시)

hintTextstring

기본 사용법​

import { ComboBox } from '@ncds/ui-admin';

const options = [
  { id: 1, label: '사과' },
  { id: 2, label: '바나나' },
  { id: 3, label: '체리' },
];

function App() {
  const [value, setValue] = useState('');
  const [filteredOptions, setFilteredOptions] = useState(options);

  const handleSearch = (searchValue: string) => {
    const filtered = options.filter((option) => option.label.toLowerCase().includes(searchValue.toLowerCase()));
    setFilteredOptions(filtered);
  };

  return (
    <ComboBox
      placeholder="과일을 검색하세요"
      optionItems={filteredOptions}
      value={value}
      onChange={setValue}
      onSearch={handleSearch}
    />
  );
}

HTML 버전​

React를 사용하지 않는 환경에서도 ComboBox를 사용할 수 있습니다.

기본 사용법​

JavaScript에서 직접 옵션을 설정할 수 있습니다.

<div id="js-combobox"></div>

<script>
  document.addEventListener('DOMContentLoaded', function () {
    const combobox = new ncua.ComboBox(document.getElementById('js-combobox'), {
      options: [
        { id: 1, label: '사과' },
        { id: 2, label: '바나나' },
        { id: 3, label: '체리' },
      ],
      value: 1, // 초기 선택값
      placeholder: '과일을 검색하세요',
      size: 'sm',
      onChange: (value) => {
        console.log('선택된 값:', value);
      },
      onSearch: (searchValue) => {
        console.log('검색어:', searchValue);
        // 여기서 옵션 필터링 로직 구현
      },
    });
  });
</script>

Multiple 선택 모드​

Multiple 선택 모드를 사용할 수 있습니다.

<div id="multiple-combobox"></div>

<script>
  document.addEventListener('DOMContentLoaded', function () {
    const multipleComboBox = new ncua.ComboBox(document.getElementById('multiple-combobox'), {
      options: [
        { id: 'react', label: 'React' },
        { id: 'vue', label: 'Vue.js' },
        { id: 'angular', label: 'Angular' },
        { id: 'svelte', label: 'Svelte' },
      ],
      value: ['react'], // 초기 선택값 (배열)
      placeholder: '프레임워크를 검색하세요',
      multiple: true, // 다중 선택 모드
      showFooterButtons: true, // 하단 버튼 표시
      size: 'sm',
      onChange: (values) => {
        console.log('선택된 값들:', values);
      },
      onSearch: (searchValue) => {
        console.log('검색어:', searchValue);
      },
      onEdit: () => {
        console.log('편집 버튼 클릭');
      },
      onSelectAll: () => {
        console.log('전체 선택 버튼 클릭');
      },
      onComplete: () => {
        console.log('완료 버튼 클릭');
      },
    });
  });
</script>

메서드​

ComboBox 인스턴스에서 사용할 수 있는 메서드들입니다.

const combobox = new ncua.ComboBox(element, options);

// 값 설정 (옵션 객체 배열 사용)
combobox.setValues([{ id: 'apple', label: '사과' }]);

// Multiple 선택 모드에서 값 설정
combobox.setValues([
  { id: 'apple', label: '사과' },
  { id: 'banana', label: '바나나' },
]);

// 현재 선택된 옵션들 가져오기 (옵션 객체 배열 반환)
const selectedOptions = combobox.getValues(); // Array<{id, label, ...}>

// 옵션 목록 가져오기
const allOptions = combobox.getOptions(); // Array<{id, label, ...}>

// 옵션 목록 업데이트
combobox.updateOptions([
  { id: 1, label: '새 옵션 1' },
  { id: 2, label: '새 옵션 2' },
]);

// 활성화/비활성화
combobox.setDisabled(true);
combobox.setDisabled(false);

// 열기/닫기
combobox.open();
combobox.close();

// 포커스 설정
combobox.focus();

// 입력 필드 초기화 (ComboBox 특화)
combobox.clearInput();

// 디버그 정보 가져오기 (ComboBox 특화)
const debugInfo = combobox.getDebugInfo();

// Multiple 모드 전용 메서드들
combobox.selectAll(); // 전체 선택
combobox.deselectAll(); // 전체 선택 해제
combobox.toggleSelectAll(); // 전체 선택 토글
const count = combobox.getSelectedCount(); // 선택된 항목 개수
const isMultiple = combobox.isMultiple(); // Multiple 모드 여부

// 컴포넌트 제거
combobox.destroy();
