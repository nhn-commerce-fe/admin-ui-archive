Examples

커스텀 스타일 적용​

const guide = stepGuide({
  el: document.body,
  options: {
    buttonLabel: {
      prev: '이전',
      next: '다음',
      done: '종료',
    },
    steps: [
      {
        title: 'Step 1',
        description: 'Step 1 description',
        element: '#name',
        position: 'bottom',
        customStyle: {
          height: '87px',
          backgroundColor: 'rgba(255, 0, 0, 0.3)',
        },
      },
    ],
  },
});

커스텀 스타일

`options.steps` 배열의 요소의 `customStyle` 속성을 추가하여 툴팁의 스타일을 커스텀할 수 있습니다.
