2026년 1월 19일

ui-admin 릴리즈 노트 (v1.4.0)​

변경일: 2025. 11. 21. (한국 표준시)

신규 기능​

  • Input 컴포넌트에 showTextCount Props를 추가하였습니다.

    • 인풋 오른쪽에 텍스트 카운트 표시 여부를 선택할 수 있습니다.

  • SelectBox 컴포넌트를 추가하였습니다.

  • ComboBox 컴포넌트를 추가하였습니다.

  • Switch 컴포넌트를 추가하였습니다.

  • FileInput 컴포넌트를 추가하였습니다.

  • ImageFileInput 컴포넌트를 추가하였습니다.

  • ui-admin-icon (dots-grid-02, flag-jp, flag-kr, flag-ch, flag-en, clock, clock-fast-forward, calendar) 업데이트했습니다.

기능 개선​

  • 보라색 계열 스타일 변수값을 추가하였습니다.

    • violet 50, 100, 200, 300, 400, 500, 600, 700

  • Button 컴포넌트의 hierarchy 수정, as 삭제 및 underline 추가하였습니다.

    • hierarchy : link 대신 text로 대체하였습니다. text-red 타입을 추가하였습니다.
    • as: 버튼의 역할만 제공하기 위해서 제거하였습니다.
    • underline: text hierarchy 외에 텍스트에 언더라인 스타일 추가되도록 props를 추가하였습니다.

  • ButtonGroup 컴포넌트에서 Button 컴포넌트 사용영역 props 업데이트건 반영하였습니다.

  • DatePicker 컴포넌트를 수정하였습니다.

    • 인풋창에 날짜와 시간을 입력할 수 있도록 수정하였습니다.
    • 시간만 설정할 수 있도록 수정하였습니다.

  • Horizontal Tab 컴포넌트를 수정하였습니다.

    • onCloseClick props를 추가하였습니다.
    • onClick props가 menus props의 객체 내부로 이동하였습니다.
    • menus props의 객체 구조를 수정하였습니다.

  • Pagination 컴포넌트에서 Button 컴포넌트 사용영역 props 업데이트건 반영하였습니다.

  • Radio, Checkbox 컴포넌트의 react-hook-form 이슈를 수정하였습니다.

    • register props를 제거하였습니다.
    • ref를 제대로 전달하지 못하는 이슈가 있어서 컴포넌트의 구조를 개선하였습니다.

  • Modal 컴포넌트를 수정하였습니다.

  • Notification 컴포넌트를 수정하였습니다.

    • autoClose 기능이 추가되었습니다.
    • supportTextLink / onHidePermanently : 링크 및 다시보지 않기 기능이 추가되었습니다.
    • full-width / info type이 추가되었습니다.

  • ProgressBar 컴포넌트의 props를 추가하였습니다.

    • valueToPercent : value를 퍼센트로 변환하는 기능이 추가되었습니다.
    • segments 옵션 : 다중 progressBar 옵션이 추가되었습니다.

  • Textarea 컴포넌트 영역의 높이가 조정 가능하도록 수정하였습니다.

  • Tooltip 컴포넌트 수정

    • positioning auto : 화면 위치에 따라 툴팁의 위치가 자동 조정 됩니다 (default: top)
    • short/long type 추가

  • Tag 컴포넌트 수정

    • tag 컴포넌트 내 글자 크기가 maxLength를 넘을 경우 마우스호버 시 툴팁형태로 노출됩니다.

문서​

  • ComboBox 컴포넌트의 가이드 문서를 추가하였습니다.
  • SelectBox 컴포넌트의 가이드 문서를 추가하였습니다.
  • FileInput 컴포넌트의 가이드 문서를 추가하였습니다.
  • ImageFileInput 컴포넌트의 가이드 문서를 추가하였습니다.
