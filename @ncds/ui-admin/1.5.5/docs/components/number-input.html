NumberInput

NumberInput은 숫자를 입력할 수 있는 입력 필드입니다. 스테퍼 버튼을 통해 값을 증가하거나 감소시킬 수 있으며, 키보드 화살표 키로도 조작할 수 있습니다. 최소값, 최대값, 증감 단위를 설정할 수 있어 정확한 숫자 입력을 제어할 수 있습니다.

기본 사용 (Basic Usage)​

스테퍼 버튼의 위치를 지정하여 기본적인 NumberInput을 사용할 수 있습니다.

크기 (Size)​

NumberInput 컴포넌트는 xs, sm 두 가지 크기로 제공됩니다.



스테퍼 위치 (Stepper Position)​

stepperPosition 속성을 사용하여 스테퍼 버튼의 위치를 설정할 수 있습니다.

스테퍼가 왼쪽에 위치합니다

스테퍼가 오른쪽에 위치합니다
스테퍼가 왼쪽에 위치합니다

스테퍼가 오른쪽에 위치합니다

최소값과 최대값 (Min & Max Value)​

minValue와 maxValue 속성을 사용하여 입력 가능한 범위를 제한할 수 있습니다. 범위를 벗어나면 스테퍼 버튼이 자동으로 비활성화됩니다.

18세 이상 100세 이하만 입력 가능합니다
18세 이상 100세 이하만 입력 가능합니다

증감 단위 (Step)​

step 속성을 사용하여 스테퍼 버튼을 클릭하거나 키보드 화살표 키를 누를 때 증감되는 값의 단위를 설정할 수 있습니다. 기본값은 1입니다.

10원 단위로 증감됩니다

1,000원 단위로 증감됩니다
10원 단위로 증감됩니다

1,000원 단위로 증감됩니다

키보드 조작​

입력 필드에 포커스가 있을 때 키보드 화살표 키(↑↓)로 값을 증감할 수 있습니다.

키보드 화살표 키로 조작해보세요
키보드 화살표 키로 조작해보세요

힌트 텍스트 (Hint Text)​

NumberInput 컴포넌트에 힌트 텍스트를 추가하여 사용자에게 추가 정보를 제공할 수 있습니다.

최소 1개 이상 주문하셔야 합니다
최소 1개 이상 주문하셔야 합니다

필수 입력 (Required)​

required 속성을 true로 설정하면 라벨 옆에 필수 표시가 나타납니다.

에러 상태 (Destructive)​

destructive 속성을 사용하여 오류 상태를 나타낼 수 있습니다.

수량을 입력해야 합니다
수량을 입력해야 합니다

유효성 검사 성공 (Validation)​

validation 속성을 사용하여 유효성 검사 성공 상태를 나타낼 수 있습니다.

올바른 수량입니다
올바른 수량입니다

비활성화 (Disabled)​

disabled 속성을 사용하여 입력 필드를 비활성화할 수 있습니다.

React Hook Form과 함께 사용하기​

NumberInput 컴포넌트는 React Hook Form과 함께 사용할 수 있습니다.

import { NumberInput } from '@ncds/ui-admin';
import { useForm } from 'react-hook-form';

function FormExample() {
  const { register, handleSubmit, setValue } = useForm();

  const onSubmit = (data) => {
    console.log(data);
  };

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <NumberInput
        label="주문 수량"
        placeholder="1"
        stepperPosition="trailing"
        minValue={1}
        maxValue={99}
        step={1}
        {...register('quantity', {
          required: '수량을 입력해주세요',
          min: { value: 1, message: '최소 1개 이상 주문하셔야 합니다' },
          max: { value: 99, message: '최대 99개까지 주문 가능합니다' },
        })}
      />
      <button type="submit">주문하기</button>
    </form>
  );
}

주요 기능​

자동 범위 제한​

  • 최소값보다 작아지면 자동으로 최소값으로 설정됩니다
  • 최대값보다 커지면 자동으로 최대값으로 설정됩니다

키보드 지원​

  • ↑ (ArrowUp): 값 증가
  • ↓ (ArrowDown): 값 감소
  • 키보드 조작 시에도 step 단위로 증감됩니다
  • 키보드 조작 시에도 min/max 범위가 적용됩니다
