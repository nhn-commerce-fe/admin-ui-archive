Switch

Switch는 사용자가 두 가지 옵션 중 하나를 선택할 수 있는 라디오 버튼 스타일의 컴포넌트입니다. 왼쪽과 오른쪽 영역을 클릭하여 선택할 수 있으며, 선택된 영역은 활성화되고 선택되지 않은 영역은 비활성화 디자인으로 표시됩니다.

인터랙티브 예제​

아래 UI에서 다양한 속성을 직접 변경하여 Switch의 모습을 확인해보세요. Left Option과 Right Option 섹션에서 각 옵션의 라벨과 값을 개별적으로 설정할 수 있습니다.

sizeselect
Switch의 크기
left.labelstring
왼쪽 옵션의 라벨
left.valuestring
왼쪽 옵션의 값
right.labelstring
오른쪽 옵션의 라벨
right.valuestring
오른쪽 옵션의 값
valuestring
Switch의 현재 선택된 값 (left.value 또는 right.value)
disabledboolean
Switch의 비활성화 여부
namestring
Switch 라디오 그룹의 name 속성
defaultValuestring
기본 선택값 (비제어 컴포넌트)

기본 사용법​

import { Switch } from '@ncds/ui-admin';

function App() {
  const [value, setValue] = useState('y');

  return (
    <Switch
      name="notification-switch"
      left={{ label: '사용함', value: 'y' }}
      right={{ label: '사용안함', value: 'n' }}
      value={value}
      onChange={setValue}
    />
  );
}

타입 정의​

SwitchSize​

type SwitchSize = 'xxs' | 'xs' | 'sm' | 'md';

SwitchOption​

type SwitchOption = {
  label: React.ReactNode; // 옵션에 표시될 라벨
  value: string | boolean; // 옵션의 값 (string 또는 boolean)
};

속성(Props)​

PropTypeDefaultDescription

namestring-라디오 그룹의 name 속성 (필수)

상세 예제​

크기 변형​

Switch는 네 가지 크기를 지원합니다. 각 크기는 NCDS 피그마 디자인 시스템을 기준으로 제작되었습니다.

기본 동작: size prop이 없거나 잘못된 값이 전달되면 자동으로 xs 크기가 적용됩니다.

비활성화 상태​

Switch를 비활성화하여 사용자 상호작용을 방지할 수 있습니다.

제어 컴포넌트와 비제어 컴포넌트​

Switch는 제어 컴포넌트와 비제어 컴포넌트 모두로 사용할 수 있습니다.

아이콘과 함께 사용​

Switch의 left.label과 right.label에 아이콘을 사용하여 더 직관적인 UI를 만들 수 있습니다.

사용 가이드​

언제 사용하나요?​

  • 이진 선택: 두 가지 상반된 옵션 중 하나를 선택할 때
  • 상태 전환: 사용/사용안함, 켜기/끄기와 같은 명확한 상태 전환
  • 즉시 적용: 선택한 옵션이 즉시 적용되는 경우
  • 시각적 명확성: 현재 선택된 상태를 시각적으로 명확하게 보여주고 싶을 때

React Hook Form과 함께 사용하기​

Switch 컴포넌트는 react-hook-form과 완벽하게 호환됩니다:

  • forwardRef 지원으로 ref 전달 가능
  • Boolean 모드와 String 모드 지원
  • Controller와 완벽한 호환성
import { useForm, Controller } from 'react-hook-form';
import { Switch } from '@ncds/ui-admin';

interface FormData {
  isEnabled: boolean;
  status: string;
}

function MyForm() {
  const { control, handleSubmit } = useForm<FormData>({
    defaultValues: {
      isEnabled: true,
      status: 'active',
    },
  });

  return (
    <form onSubmit={handleSubmit(console.log)}>
      {/* Boolean 모드 - boolean 값 직접 사용 */}
      <Controller
        name="isEnabled"
        control={control}
        render={({ field }) => (
          <Switch
            {...field}
            left={{ label: '사용함', value: true }}
            right={{ label: '사용안함', value: false }}
            name="isEnabled"
          />
        )}
      />

      {/* String 모드 - 실제 string 값 */}
      <Controller
        name="status"
        control={control}
        render={({ field }) => (
          <Switch
            {...field}
            left={{ label: '활성', value: 'active' }}
            right={{ label: '비활성', value: 'inactive' }}
            name="status"
          />
        )}
      />
    </form>
  );
}

자동 타입 감지​

Switch 컴포넌트는 left.value와 right.value의 타입을 자동으로 감지합니다:

  • Boolean 모드: 두 값이 모두 boolean 타입일 때 자동 활성화
  • String 모드: 하나 이상의 값이 string 타입일 때 사용

Ref 사용하기​

개별 radio input에 접근해야 하는 경우 refs 배열을 사용할 수 있습니다:

import { useRef } from 'react';
import { Switch } from '@ncds/ui-admin';

function MyComponent() {
  const leftInputRef = useRef<HTMLInputElement>(null);
  const rightInputRef = useRef<HTMLInputElement>(null);

  const handleFocusLeft = () => {
    leftInputRef.current?.focus();
  };

  const handleFocusRight = () => {
    rightInputRef.current?.focus();
  };

  return (
    <Switch
      left={{ label: '활성', value: 'active' }}
      right={{ label: '비활성', value: 'inactive' }}
      refs={[leftInputRef, rightInputRef]}
      name="example"
    />
  );
}

접근성 고려사항​

  • Switch는 내부적으로 radio input을 사용합니다
  • 라디오 버튼은 숨겨져 있으며 시각적 표시는 label 요소로 구현됩니다
  • checked 속성으로 현재 상태를 스크린 리더에 전달합니다
  • name 속성을 통해 라디오 그룹을 형성하고 폼 데이터와 연동됩니다
  • ref를 통해 실제 radio input 요소에 접근할 수 있습니다
