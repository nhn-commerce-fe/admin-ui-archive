DatePicker

DatePicker 컴포넌트는 사용자가 날짜를 선택할 수 있는 입력 컴포넌트입니다. flatpickr 라이브러리를 기반으로 하며, 한국어 로케일을 지원합니다. 다양한 크기 옵션과 오류 상태 표시 기능을 제공합니다.

인터랙티브 예제​

아래 UI에서 다양한 속성을 직접 변경하여 DatePicker 컴포넌트의 모습을 확인해보세요.

현재 선택된 날짜 (YYYY-MM-DD 형식)

자동 포커스 여부

기본 사용법​

import { DatePicker } from '@ncds/ui-admin';
import { useState } from 'react';

function App() {
  const [date, setDate] = useState('2024-03-20');

  return <DatePicker currentDate={date} onChangeDate={setDate} />;
}

Props​

PropTypeDefaultDescription
size'xs' | 'sm''xs'DatePicker의 크기를 지정합니다.
shouldFocusbooleantrue컴포넌트 마운트 시 자동 포커스 여부를 지정합니다.
currentDatestring필수

현재 선택된 날짜. 형식은 datePickerOptions의 dateFormat에 따라 다릅니다.

  • 날짜만: 'YYYY-MM-DD' 형식
  • 날짜 + 시간: 'YYYY-MM-DD HH:mm' 형식
  • 날짜 + 시간 + 초: 'YYYY-MM-DD HH:mm:ss' 형식
  • 시간만: 'HH:mm' 또는 'HH:mm:ss' 형식 빈 문자열('')을 전달하면 초기값 없이 표시됩니다.
disabledbooleanfalseDatePicker를 비활성화합니다.
autocomplete'on' | 'off''off'자동 완성 허용 여부 (기본 off)
destructivestring-오류 메시지를 넣습니다.
placeholderstring-placeholder 값을 넣습니다.
isEndDateboolean-

종료일자로 지정됩니다. isEndDate:true 이고 enableTime, enableSeconds가 true로 설정 되면 시간이 따로 지정한 값이 없다면 23:59:59로 설정 됩니다.


만일 isEndDate가 false또는 없고 enableTime, enableSeconds가 true로 설정될 경우 시간은 따로 지정한 값이 없다면 00:00:00으로 설정 됩니다.


enableSeconds가 false이고 enableTime이 true인 경우, isEndDate에 따라 23:59 또는 00:00으로 설정됩니다.

datePickerOptionsOptions-

flatpickr의 추가 옵션을 지정합니다. 주요 옵션:

  • enableTime: 시간 선택 활성화
  • enableSeconds: 초 단위 선택 활성화
  • noCalendar: 캘린더 숨기기 (시간만 선택)
  • minDate/maxDate: 최소/최대 날짜 제한
  • dateFormat: 날짜 형식 지정
  • allowInput: 직접 입력 허용
  • disableMobile: 모바일 기본 날짜 선택기 비활성화 자세한 옵션은 flatpickr 문서를 참조하세요.
classNamestring-DatePicker 컨테이너에 추가할 CSS 클래스
onChangeDate(date: string) => void필수

날짜가 변경될 때 호출되는 콜백 함수. 선택된 날짜를 문자열 형식으로 전달합니다.

  • 날짜만 선택 시: 'YYYY-MM-DD' 형식
  • 날짜 + 시간 선택 시: 'YYYY-MM-DD HH:mm' 형식
  • 날짜 + 시간 + 초 선택 시: 'YYYY-MM-DD HH:mm:ss' 형식
  • 시간만 선택 시: 'HH:mm' 또는 'HH:mm:ss' 형식
onValidationError

(error: { date: Date; minDate?: any; maxDate?: any; violations: Array<'minDate' | 'maxDate'>; previousDate?: Date; }) => void

-

minDate/maxDate 범위를 벗어난 날짜가 선택되었을 때 호출되는 콜백 함수입니다.
- date: 선택된 날짜 (범위를 벗어난 날짜)
- minDate: 설정된 최소 날짜
- maxDate: 설정된 최대 날짜
- violations: 위반된 제약 조건 배열 ('minDate' 또는 'maxDate')
- previousDate: 이전에 선택된 날짜 (복원 시 사용)
onValidationError가 제공되면 외부에서 validation을 처리할 수 있으며, 제공되지 않으면 범위를 벗어난 날짜도 허용됩니다.

DatePicker Component argument

속성타입필수설명

optionsDatePickerOptions✓

DatePicker 설정 옵션 객체. 다음 속성들을 포함합니다:

  • size: 'xs' | 'sm' (DatePicker 크기)
  • separator: string (날짜 구분자, 기본값: '~')
  • buttons: DatePickerButton[] (기간 선택 버튼 배열, 선택사항)
  • datePickerOptions: DatePickerProps[] (DatePicker 설정 옵션 배열)
  • onValidationError: (error: ValidationError) => void (유효성 검사 오류 콜백, 선택사항)

속성타입필수설명

flatpickr 옵션. 주요 옵션:

  • mode: 'single' | 'range' (날짜 선택 모드)
  • enableTime: boolean (시간 선택 활성화)
  • enableSeconds: boolean (초 단위 선택 활성화)
  • noCalendar: boolean (캘린더 숨기기, 시간만 선택)
  • dateFormat: string (날짜 형식, 예: 'Y-m-d', 'Y-m-d H:i', 'Y-m-d H:i:S')
  • minDate/maxDate: string (최소/최대 날짜 제한)
  • allowInput: boolean (직접 입력 허용)
  • disableMobile: boolean (모바일 기본 날짜 선택기 비활성화)
  • onChange: (selectedDates, dateStr, instance) => void (날짜 변경 콜백) 자세한 옵션은 flatpickr 문서를 참조하세요.

<script>
  const today = new Date();
  document.addEventListener('DOMContentLoaded', function () {
    const datePicker = new ncua.DatePicker(document.querySelector('#datepicker-container'), {
      size: 'xs',
      datePickerOptions: [
        {
          element: 'start-date',
          placeholder: '시간 입력' // datePicker.setDate(['']); 빈 값 설정 해야 노출 됩니다.
          attrName: 'hello', // <input name="hello" type="text" />
          options: {
            mode: 'single',
            static: true,
            defaultDate: today,
            dateFormat: 'Y-m-d', // 시간을 표시 하려면 'Y-m-d H:i'값으로 수정, 초까지 표시하려면 'Y-m-d H:i:S' (flatpickr 옵션)
            enableTime: true,  // 시간 표시 할 때 옵션 추가시 시간 표시될 때 input 너비 수정 됨 (flatpickr 옵션)
            enableSeconds: false, // 초 단위 선택 활성화 (flatpickr 옵션)
            clickOpens: true,
            allowInvalidPreload: true,
            locale: 'ko',
            onChange: (selectedDates, dateStr, instance) => {
              console.log(selectedDates, dateStr, instance);
              startDate = dateStr;
            },
          },
        },
      ],
    });

    // datepicker 날짜 가져오기
    console.log(datePicker.getDates());

    // datepicker 날짜 설정하기
    datePicker.setDate(['2025-07-01']);
  });
</script>

메서드​

getDates()

선택된 날짜들을 배열로 반환합니다.

const dates = datePicker.getDates();
// 예: ['2025-07-01', '2025-07-31']
// 날짜가 선택되지 않은 경우 빈 문자열('')이 반환됩니다.

setDate(dates: string[])

DatePicker의 날짜를 설정합니다.

// 단일 날짜 설정
datePicker.setDate(['2025-07-01']);

// 범위 날짜 설정 (Range DatePicker인 경우)
datePicker.setDate(['2025-07-01', '2025-07-31']);

// 빈 값으로 설정 (placeholder 표시)
datePicker.setDate(['']);

옵션​

onValidationError

날짜 유효성 검사 오류 발생 시 호출되는 콜백 함수입니다.

const datePicker = new ncua.DatePicker(wrapper, {
  // ... 기타 옵션
  onValidationError: (error) => {
    const { index, date, minDate, maxDate, violations, previousDate } = error;
    console.log('오류 발생:', violations); // ['minDate'] 또는 ['maxDate']
    console.log('이전 날짜:', previousDate);
    console.log('현재 날짜:', date);
    
    // 필요시 이전 값으로 복원
    if (previousDate) {
      const currentDates = datePicker.getDates();
      currentDates[index] = previousDate;
      datePicker.setDate(currentDates);
    }
  }
});
