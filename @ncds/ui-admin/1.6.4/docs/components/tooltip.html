Tooltip

툴팁은 요소에 대한 추가 정보를 표시하는 작은 팝업 창입니다. 사용자가 요소에 마우스를 올리거나 포커스할 때 나타납니다.

인터랙티브 예제​

아래 UI에서 다양한 속성을 직접 변경하여 Tooltip 컴포넌트의 모습을 확인해보세요.

툴팁 타입 예제​

Short 타입 (기본)​

Short 타입은 마우스 호버로 표시되고 마우스 아웃으로 숨겨지는 기본 툴팁입니다.

Short 툴팁마우스 호버로 표시됩니다
Short 툴팁마우스 호버로 표시됩니다

Long 타입​

Long 타입은 마우스 호버로 표시되지만 호버 아웃해도 사라지지 않고, X 버튼으로만 닫히는 툴팁입니다. 긴 텍스트를 표시할 때 사용합니다.

Long 툴팁마우스 호버로 표시되고, X 버튼으로 닫을 수 있습니다. 긴 텍스트도 여러 줄로 표시됩니다.
Long 툴팁마우스 호버로 표시되고, X 버튼으로 닫을 수 있습니다. 긴 텍스트도 여러 줄로 표시됩니다.

타입별 비교​

Short:Short호버로 표시
Long:Long호버로 표시하고 X 버튼으로 닫기. 이렇게 긴 텍스트도 표시할 수 있습니다.

HTML 버전 예제​

JavaScript 버전에서도 동일한 기능을 사용할 수 있습니다:

<!-- Short 타입 (기본) -->
<script>
  const shortTooltip = window.ncua.Tooltip.createShort('Short 툴팁', '마우스 호버로 표시됩니다');
  document.getElementById('container').appendChild(shortTooltip.getElement());
</script>

<!-- Long 타입 -->
<script>
  const longTooltip = window.ncua.Tooltip.createLong(
    'Long 툴팁',
    '마우스 호버로 표시되고, X 버튼으로 닫을 수 있습니다.'
  );
  document.getElementById('container').appendChild(longTooltip.getElement());
</script>

<!-- 옵션으로 생성 -->
<script>
  const customTooltip = window.ncua.Tooltip.create({
    type: 'long',
    tooltipType: 'white',
    iconType: 'fill',
    position: 'auto',
    title: '커스텀 툴팁',
    content: '다양한 옵션을 설정할 수 있습니다.',
  });
  document.getElementById('container').appendChild(customTooltip.getElement());
</script>

주의사항​

  1. 툴팁은 viewport를 벗어나지 않도록 자동으로 위치가 조정됩니다.
  2. content 속성은 React.ReactNode 타입으로 문자열뿐만 아니라 React 요소(태그)도 사용할 수 있습니다.
