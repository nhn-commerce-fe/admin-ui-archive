Image File Input

ImageFileInput 컴포넌트는 사용자로부터 파일을 업로드하기 위한 UI 요소입니다. 이 컴포넌트는 다양한 파일 타입을 지원하며, 파일 크기 제한, 개수 제한 등의 기능을 제공합니다.

타입 정의​

ImageUploadedFile​

서버에서 받은 기존 파일 정보를 나타내는 타입입니다.

type ImageUploadedFile = {
  fileName: string; // 파일명
  fileImageUrl: string; // 파일 URL 또는 경로
};

다중 파일 + 유효성 검사​

방법 1: container 옵션 사용​

const fileInput = new ncua.ImageFileInput({
  container: 'my-container', // 자동으로 추가
  label: '이미지 업로드',
  buttonLabel: '이미지 선택',
  multiple: true,
  accept: 'image/*',
  maxFileCount: 3,
  maxFileSize: 2 * 1024 * 1024, // 2MB
  hintItems: ['최대 3개의 이미지 파일을 업로드할 수 있습니다.', '각 파일은 최대 2MB까지 업로드 가능합니다.'],
  onFileSelect: (files) => {
    console.log('선택된 파일:', files);
  },
  onFail: (invalidFiles) => {
    console.error('업로드 실패:', invalidFiles);
  },
});

Controlled Mode​

let currentFiles = [];

const fileInput = new ncua.ImageFileInput({
  label: 'Controlled 파일 입력',
  buttonLabel: '파일 찾기',
  multiple: true,
  value: currentFiles,
  onChange: (files) => {
    currentFiles = files;
    console.log('현재 파일:', currentFiles);
  },
});

// 프로그래밍 방식으로 제어
fileInput.clearFiles(); // 모든 파일 제거
fileInput.setFiles([{ fileName: 'file.pdf', fileImageUrl: '/uploads/file.pdf' }]); // 초기 파일 설정 (UploadedFile[])
const files = fileInput.getFiles(); // 파일 조회
